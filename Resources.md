# Resources for continued learning!
Here is a list of resources for continued learning in programming and dealing with data! If you're unsure of which language to go for, talk to your labmates. If you're not in a lab yet, look through some recent (within the last 3 years) papers from groups you'd want to work with and see what they're using for data processing and analysis. 

When looking for a tutorial/course, it helps to be specific. You can use programming for lots of things: writing software, making video games, analyzing data, hacking into the mainframe in a 90s movie, etc...but a lot of what we use in science fits into 'data science'. As you're looking around, start with courses that offer programming for research/programming for data science/scientific programming. 

## R(Studio)
R is an open-source programming language, and RStudio is its development environment (for more on environments, see [Session_0](https://github.com/INN-SFU/SummerResearchBootcamp/blob/main/Session_1/Session_0.ipynb)). It's commonly used in statistics courses at the undergraduate and graduate level and is freeeeeee! It shares similarities with the other languages in this list in how it looks (its user interface looks like MatLab and it uses base 1 indexing) and how it acts (its package structure and syntax are very similar to python)
1. [R for Data Science](https://r4ds.hadley.nz/)
2. [Introduction to R on Coursera](https://www.coursera.org/courses?query=r): Coursera provides good programming courses, but these sometimes cost money (it's possible to get a fee waiver, but it does seem a little random)


## Python
Python is an open-source programming language. It's commonly used in neuroimaging, machine learning, and is growing in popularity for statistical analyses and processing large datasets. It uses base 0 indexing and has many possible configurations. It is also freeeeeeeee!
1. [Cognitive Class Applied Data Science](https://cognitiveclass.ai/learn/applied-data-science-with-python)
2. [Software Carpentry: Programming with Python](https://swcarpentry.github.io/python-novice-inflammation/)
3. [100 Days of code](https://www.reddit.com/r/replit/comments/1s8ihbo/replit_100_days_of_code/): RIP, this was a great intro to Python. Reddit users are assembling the material they can find into an unlisted YouTube playlist linked in the thread

## Matlab
Matlab is NOT an open-source programming language, but you can get a free license from SFU ([instructions here] (https://www.sfu.ca/information-systems/services/software/matlab/matlab-install.html)). Matlab is still widely used in neuroimaging research and is particularly well-suited for dealing with large datasets. It has lots of built-in functionality (unlike R and Python's package structure), but is becoming less popular due to its cost.
1. [Introduction to Matlab on Coursera](https://www.coursera.org/learn/matlab): as above, but the lectures are (currently) [all on YouTube](https://www.youtube.com/playlist?list=PL9U0RhCnVfG4MJqTPwdKiSTnNuH7wEIA2)

SFU's Research Computing Group are great to work with and have many workshops throughout the year.
1. [Check them out here](https://training.westdri.ca/events/upcoming-training-winter-spring-2026/)

## Fun Stuff
1. [Awesome Public Datasets](https://github.com/awesomedata/awesome-public-datasets): want to play with more data but don't have anything to work with? No worries - the good people of The Internet are here for you!
2. [Spurious Correlations](https://www.tylervigen.com/spurious-correlations): a website of statistically significant correlations...that do not (thankfully) equal causation
