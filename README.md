# INN Summer Research Bootcamp
## Technical Skills Sessions
Here is where you will find the notebooks, slides, and data for the technical workshops as part of the 2026 INN Summer Research Bootcamp

## Undergraduate Sessions
The undergraduate sessions will be run using notebook (.ipynb) files. If you have a prefered way of running python notebooks, download the notebook and data and open it up in your environment (or pull it directly from github). 

If you don't, great! You have two options:
1. If you're a google user with a google account, you can use [collab notebooks](https://colab.research.google.com/). You will need to be signed in to run the code

2. If you're not a google use, you can use [Jupyter online](https://jupyter.org/try-jupyter/tree/)
  There are a couple extra steps for Jupyter online users! You'll have to download the data file (FilmDataStart.csv in the 'Session 1' folder) and upload it to the notebook, but then you'll be good to go!
  
    When it asks what server you want to use, select 'Python (Pyodide)'
